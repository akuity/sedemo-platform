# Templatized Kargo Project

This project(s) demonstrates how a team can create a golden path for application teams to rapidly adopt.  The single helm chart defines a standard flow, allowing individual application teams to modify exposed parameters to customize but stay within guardrails. 


- IaC Defined roles for Dev, SRE, and QA teams with unique permissions. 
- PR based approvals for Prod deploy
- Rollouts with Canary deploy


THe "team" names are mostly identical, and only exist to highlight that we can create many teams/projects from a single template.
